La Bête à Corne

Comment faire une Bête à Corne

La bête à corne est un outil d'analyse fonctionnelle qui permet de définir le besoin d'un produit ou d'un service.

  1. Identifier le produit ou service
  2. Définir les utilisateurs
  3. Identifier les besoins
  4. Établir les relations entre ces éléments

Exemple de Bête à Corne

La Pieuvre

Comment faire une Pieuvre

La pieuvre est un diagramme qui permet d'identifier les fonctions principales et les contraintes d'un produit.

  1. Identifier le produit central
  2. Lister les fonctions principales
  3. Identifier les contraintes
  4. Établir les relations

Exemple de Pieuvre

Le Diagramme de Pareto

Comment faire un Pareto

Le diagramme de Pareto est un outil qui permet d'identifier les causes les plus importantes d'un problème.

  1. Collecter les données
  2. Trier les données par ordre décroissant
  3. Calculer les pourcentages cumulés
  4. Représenter graphiquement
